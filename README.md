# AirPnP

Web application for renting parking places around town for people who may have parking slots available, similar to AirBnB.

Application is built on top of the Java Spring Boot MVC framework. User interface is rendered on server side using JSP - Standard Tag Library (JSTL). Backend performs objectâ€“relational mapping to local H2 database using Hibernate.

## Build and run

From the terminal run `mvn clean install`
